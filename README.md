# nome-do-repositório-e-teste-lehnert-calculadora-básica
Um projeto prático desenvolvido para consolidar conceitos fundamentais de lógica de programação. A ferramenta realiza operações básicas (soma, subtração, multiplicação e divisão) via terminal, com foco em código limpo e tratamento de erros simples. Ideal para quem está iniciando no ecossistema Python.
